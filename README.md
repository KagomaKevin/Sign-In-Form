Sign In Form (HTML & CSS)
A simple, clean, and responsive Sign In (Login) Form built using HTML and CSS only.
This project focuses purely on frontend UI design, layout, and responsiveness.
Features
• Clean and modern UI
• Fully responsive design
• Email and password inputs
• Icons inside input fields
Technologies Used
HTML5, CSS3
Project Structure
index.html – Form structure
style.css – Styling
README.md – Documentation
How to Run
1. Download or clone the project
2. Open index.html in a browser
Customization
You can change colors, icons, fonts, and layout easily using CSS.
Future Improvements
• JavaScript validation
• Backend integration
• Dark mode
